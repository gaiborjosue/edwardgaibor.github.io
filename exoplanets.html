Exoplanet Prediction Algorithm with a 92% accuracy score.

As the creator of this project, my role was:

=> Download data from NASA's online database. Then process and clean them with Python tools (Pandas, Numpy).

=> Create graphs to understand the tendencies of the data and analyze the possible future relationship between the variables for the algorithm with Jupyter.

=> Create the algorithm's model with the Decision Tree Classifier library. Save the Model in a Joblib file.

=> Train the Model from the Joblib file in Python with the processed data and evaluate its accuracy score.

=> Add Parse Arguments so that users can use the Model from the CLI.

=> Publish it with GIT.
